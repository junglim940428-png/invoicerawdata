새 인보이스 파일 추가 방법

📁 GitHub 저장소 → raw_data/ 폴더에 xlsx 파일 업로드 ↓ (자동, 약 1~2분) ⚙️ GitHub Actions 실행 → 중복 제거 + 누적 처리 ↓ (자동) 💾 data/db.json 업데이트 → GitHub Pages 반영 ↓ (즉시) 👥 팀원들이 이 페이지 새로고침 → 최신 데이터 검색 가능

GitHub 웹에서 파일 업로드하는 법:
저장소 → raw_data/ 폴더 클릭 → 우측 상단 Add file → Upload files → xlsx 파일 드래그 → Commit changes

2
중복 처리 기준

SHIPPING_NO 컬럼 기준으로 중복을 자동 제거합니다.
같은 SHIPPING_NO가 여러 파일에 있으면 처음 등장한 것만 남기고 나머지는 제거됩니다.
파일을 여러 번 올려도 중복 걱정 없이 안전합니다.

3
주의사항

• 이 툴은 GitHub Pages로 제공되므로 저장소가 Public이어야 무료로 사용 가능합니다.
• S/N 데이터가 외부에 노출되어도 무방한지 사전에 확인하세요.
• 인터넷 연결이 필요합니다 (사내 인트라넷 전용 환경이면 사용 불가).
